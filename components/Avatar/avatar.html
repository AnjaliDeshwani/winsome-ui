Avatar

Avatars are the profile icons

Image Avatar

To include image avatar you need to include 'avatar' class and for different sizes you can include 'avatar-lg', 'avatar-md', 'avatar-sm'.

Text Avatar

Text avatar is used at places where profile pic of user is missing.
To include text avatar you nedd to include 'avatar-text'.
